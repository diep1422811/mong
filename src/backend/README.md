# Dev

1. Cài môi trường ảo, dùng python 3.10

   - Chạy lệnh `python -m venv .venv`
   - Chạy lệnh `source .venv/bin/activate`
   - Chạy lệnh `pip install -r requirements.txt`

2. Chạy server
   - Dùng docker để chạy: `docker compose up --build`
